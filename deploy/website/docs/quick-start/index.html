快速开始

三步搞定：登录拿到免费积分，接入一个聊天 App，写第一条规则。全程不超过 5 分钟。

第一步：登录拿免费积分

打开管理面板，右上角点「登录 / 注册」。用邮箱注册即可，新用户自动获得 100 注册积分 + 每日 100k tokens 免费额度。 登录后默认走 OpenRouter 代理 —— 不需要你自己填 API key 也能立刻开聊。

不想用云积分？ 可以在「设置 → 接入模式」里切到「自托管」，自己填 OpenAI / Anthropic / DeepSeek 等服务商的 API key，完全本地运行。

第二步：接入一个聊天 App

左侧导航点「渠道」，选一个你常用的平台。下面以 Telegram 为例：

  1. 找 Telegram 的 @BotFather，/newbot 创建一个新 bot，拿到 token
  2. 在 DlxAI 渠道页选「Telegram」，粘贴 token，保存
  3. 在 Telegram 里搜你刚创建的 bot，发一句话试试

其他平台流程类似 —— 详见 聊天渠道。

第三步：写第一条规则

左侧导航点「规则」，输入框里用大白话写一句：

每天早上 9 点，把昨天 Telegram 里收到的所有消息总结成一段话发给我

保存。规则会自动编译为 cron 任务 + 总结 skill，立即生效，不用重启。

就这样

现在你的 DlxAI 已经在 Telegram 里 24/7 待命。下一步可以：

  • 接入更多渠道（飞书、Discord、Slack...）—— 同一个 agent 共享上下文
  • 读 自然语言规则 学习更复杂的写法
  • 逛 技能市场 一键安装社区写好的 skill
