LLM 服务商

DlxAI 是 LLM 厂商中立的 —— 你可以用云积分（默认走 OpenRouter），也可以填自己的 API key 用任何主流模型。

三种接入模式

  • 积分模式（默认）：登录就能用，DlxAI 后端代付 OpenRouter 调用费，按 token 扣积分
  • 自托管：自己填各服务商的 API key，完全本地，零云依赖
  • 订阅 / 编程计划：连接 Claude / Gemini / Kimi 等服务商的订阅账号，免 API key

添加一个 API key

  1. 左侧导航点「服务商」
  2. 选你要用的服务商，点「添加 key」
  3. 粘贴 key（会通过 Keychain / DPAPI 加密存储）
  4. 选默认模型
  5. 保存 —— 立即生效

按服务商配置代理

在受限地区？每个服务商或每把 API key 都可以单独配 HTTP / SOCKS5 代理。 DlxAI 的 proxy router 会自动按 host 路由流量，不影响其他流量。

热重载： 改 API key、加新服务商、改代理 —— 全部即时生效，不需要重启 gateway。
