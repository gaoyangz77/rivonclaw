欢迎使用 DlxAI

DlxAI 是一个本地优先的个人 AI 助理。安装一次，就能在 Telegram、Discord、Slack、飞书、iMessage 等十几个聊天 App 里找到它。 数据全部留在你自己的机器上，规则用大白话写，不需要写代码也不需要画流程图。

核心理念： OpenClaw 是引擎，DlxAI 是驾驶舱 —— 把 OpenClaw 这套强大但工程师向的 agent 运行时，包装成人人都能用的桌面应用。

为什么选 DlxAI

  • 个人，而非客户向：不像 Botpress 或 Voiceflow 给企业建客服 bot，DlxAI 是你的私人 AI，跟随你的身份和偏好。
  • 主动来找你：不像 Jan、Msty 等本地聊天客户端要你打开 app，DlxAI 在你常用的聊天 App 里收发消息。
  • 大白话写规则：不像 n8n 或 Make 那样要画流程图，写一句话就能定义行为，立即生效。
  • 本地优先：数据存本地 SQLite，密钥用 macOS Keychain / Windows DPAPI 加密，永不上传。
  • 多 LLM 中立：20+ 服务商任选 + 内置免费积分，零配置开聊。

下一步

  • 安装 DlxAI —— Mac / Windows / Linux 一键安装包
  • 快速开始 —— 三步从安装到第一次对话
  • 配置聊天渠道 —— Telegram、Discord、Slack 等
  • 写第一条规则 —— 用自然语言定义 agent 行为
