常见问题

它到底是什么？

DlxAI 是一个本地优先的桌面 AI 助理。它把 OpenClaw 这套强大的 agent 运行时包装成桌面 app， 并且接入了十几个聊天平台 —— 你可以在 Telegram、Discord、Slack、飞书等任何一个常用的聊天 App 里召唤它。

和 ChatGPT / Claude desktop 有什么区别？

最大区别是：ChatGPT 和 Claude 是 chat 客户端，你打开它跟 LLM 说话。 DlxAI 是 agent，它住在你已经在用的聊天 App 里，主动帮你完成任务（提醒、总结、回复客户...）。 而且 DlxAI 数据在本地，你可以自己换 LLM 服务商。

需要 API key 吗？

不需要。新用户登录后默认走 DlxAI 云端积分（每日 100k tokens 免费），开箱即用。 如果想用自己的 API key，去「设置 → 接入模式」切换到「自托管」即可。

数据安全吗？

DlxAI 是本地优先的：

  • 所有对话历史、规则、技能都存在本地 SQLite
  • API key 和 token 通过 macOS Keychain / Windows DPAPI 加密
  • 除了你主动调用的 LLM 服务商，没有任何数据上传
  • 开源代码可审计

支持哪些操作系统？

macOS 12+、Windows 10/11、Ubuntu 22.04+。三平台都有官方安装包。

渠道排查：消息收不到 / 发不出去

  1. 左侧「日志」面板看 gateway 输出有没有报错
  2. 「渠道」页确认对应账号状态是「已连接」
  3. 检查 token 是否过期 / 被吊销
  4. 检查群组是否在 allowlist 里（默认只响应白名单群）
  5. 对于代理依赖的渠道（Telegram 在中国大陆），确认服务商代理已配置

我能在公司服务器上跑吗？

可以。DlxAI 是 Electron app，但 panel-server 只是个 HTTP 服务，可以脱离 Electron 单独部署。 不过这种用法目前没有官方文档支持，建议先用桌面客户端跑通流程再考虑迁移。

它会代替 n8n / Botpress / Zapier 吗？

定位完全不同。n8n / Zapier 是 workflow 自动化（你画流程图），Botpress 是企业客服 bot 平台。 DlxAI 是 个人 AI agent —— 它服务的是你本人，而不是你的客户或一套业务流程。

开源协议？

MIT。代码托管在 GitHub，欢迎 issue 和 PR。
